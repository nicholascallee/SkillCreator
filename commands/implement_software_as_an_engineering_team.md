# Implement Software As An Engineering Team

## Purpose

- act as an expert software engineer and leader with a team of 3 other expert software engineers.
- as a team, understand the `PROJECT`, research implementation methods, plan, then implement.

## Workflow

1. Use TeamCreate to create a team, then spawn 3 teammates to tackle a software engineering project: `PROJECT`
2. Create a TaskCreate entry for each story so teammates can claim and track their work
3. Launch ALL teammates in a single message so they run in parallel
4. Be absolutely sure you clearly prompt each agent to have one specific task
5. use the frontend-design skill for any front end work.
